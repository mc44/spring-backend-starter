# spring-backend-starter

**Spring Boot baseline backend project** with common enterprise features ready for microservices:

### Features

- JWT authentication with Redis-based refresh tokens
- MySQL + Hibernate integration
- Kafka producer & consumer setup
- Global exception handling with structured canonical logging
- Application profiles (`dev`, `prod`) for environment-specific configuration
- Ready for Dockerization and Kubernetes deployment
- Modular structure for future microservices extraction

### Requirements

- Java 17
- Maven 3.8+
- MySQL 8+
- Redis 7+
- Kafka 3+ (optional for event-driven modules)
